1.Spring Boot Security basics

To enable security to spring boot project
- Update pom with security artifact
	<dependency>
		<groupId>org.springframework.boot</groupId>
		<artifactId>spring-boot-starter-security</artifactId>
	</dependency>

- in application.properites enter the following properites (for custom credentails)
	security.user.name=admin
	security.user.password=admin 

	default username and password is user/16-digit UUID
